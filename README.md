## 📌 Problem Statement

Coffee plants are highly vulnerable to diseases like:
- Rust
- Leaf Miner
- Nutritional Deficiencies

Manual detection is:
- Time-consuming
- Requires expertise
- Not scalable for farmers

This project provides an AI-based automated solution.

---

## 🧠 Model Details

- Model Type: Convolutional Neural Network (CNN)
- Input Size: 224x224 RGB images
- Classes:
  - Healthy
  - Leaf Miner
  - Rust
- Output: Softmax probability scores

The model was trained on a labeled coffee leaf dataset and optimized for classification accuracy.

---

## 🖥️ Features

✔ Upload coffee leaf image  
✔ Disease prediction with confidence score  
✔ Probability distribution chart  
✔ Farmer recommendation system  
✔ Multi-language support (English & Kannada)
