## 📌 Project Overview

This project is an AI-powered Coffee Leaf Disease Detection system built using **TensorFlow, MobileNetV2 (Transfer Learning), and Streamlit**.

The system classifies coffee leaf images into:

- 🌿 Healthy
- 🍂 Leaf Miner
- 🍁 Rust

It also includes:

✔ Grad-CAM (Explainable AI visualization)  
✔ Bilingual Support (English & Kannada)  
✔ Cause & Prevention Recommendations  
✔ Downloadable PDF Report  
✔ Deployment using Streamlit Cloud  

---

## 🧠 Key Features

### 🔍 1. Deep Learning Model
- Transfer Learning with **MobileNetV2**
- Functional API architecture
- EarlyStopping for performance optimization

### 🔬 2. Explainable AI (Grad-CAM)
- Visual heatmap highlighting infected regions
- Helps understand model decision-making
- Makes system interview & research ready

### 🌍 3. Multi-Language Support
- English
- Kannada (Farmer-friendly)

### 🌿 4. Agricultural Advisory
Provides:
- Disease cause
- Prevention methods
- Farmer recommendations

### 📄 5. Downloadable PDF Report
- Prediction
- Confidence score
- Cause & Prevention guidance
